# The Relocation Briefcase

A reusable evaluation system for relocation candidates, built in the **Due West** visual identity.

**Live:** https://jerrunge.github.io/relocation/

## Structure

- **`index.html`** — the hub. Holds *The Fifteen* (the destination-agnostic rubric) and *The Candidates* index. Every place gets graded on the same fifteen decisions so none gets scored on charm instead of fit.
- **`grass-valley-nevada-city.html`** — Candidate **A1** (California / Lanterman entitlement). Chart-map, full scorecard, trails (AllTrails), restaurants, and culture.

Add a destination by copying the A1 page, re-scoring against The Fifteen, and adding a row to the Candidates index.

## The gate

Only two states clear the non-negotiable — Maddy's support services: **California** (Lanterman Act entitlement) and **Oregon** (K-Plan, no waitlist). Every candidate lives in one of those two.

## Design

Due West system (locked 2026-07-03). Schibsted Grotesk / Newsreader / Instrument Sans / IBM Plex Mono. Paper ground, dusk cover bands, kelp / rust / slate category accents, Signal Red reserved for the 270° course line.

## Sources

Regional-center research (ACRC / DDS), Redfin, Movoto, National Geographic (2025), AllTrails, Google Places, and on-the-ground scout notes. Every mark is real.
